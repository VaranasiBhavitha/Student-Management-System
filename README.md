# 🎓 Student Management System (Java)

## 📌 Description
This is a simple Java-based Student Management System that allows users to manage student records using basic operations.

## 🚀 Features
- ➕ Add Student
- 📋 View Students
- 🔍 Search Student by ID
- ❌ Delete Student

## 🛠️ Technologies Used
- Java
- OOP (Object-Oriented Programming)
- ArrayList

## ▶️ How to Run

1. Compile the program:
   javac StudentManagement.java

2. Run the program:
   java StudentManagement
